# Qu'est-ce qu'un agent ?

Plusieurs définitions :
* systèmes <span v-mark.circle.red="1">complètement autonomes</span> qui sont en mesure d'utiliser des outils externes pour accomplir des tâches
* un enchaînement d'actions prédéterminées suivant un <span v-mark.circle.red="2">workflow</span> bien définis.

Anthropic utilise le concept de <span v-mark.underline.blue="3">système d'agents (agentic systems)</span>
